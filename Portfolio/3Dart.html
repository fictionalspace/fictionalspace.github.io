Plume of Luxury: 3D Printed Edition

Medium: Digital Art -created in Minecraft and 3D printed

Plume of Luxury is an artifact item from the action RPG Genshin Impact.
Using Minecraft to build up a pixel model of Plume of Luxury and porting the model through the software Mineway
to convert my Minecarft creation model into a STL file. Finally using a 3D printer to input the file into the machine
to start printing out the craft. The first prototypes, the small black plastic wired ones didn't work out how I would like
since the pixel looking diagonal line is suppose to be like dangling beads which didn't translate through. For my 2nd prototype
I went back to my Minecreaft creation to separate the beads from the main piece, ported once more through Mineway, and altered
the size dimensions for the new STL file. Inserting the file to be printed again in the 3D printer, only finally did my piece
came out the way it was intended. Lastly I used acrylic paint to add in the finishing touches.
