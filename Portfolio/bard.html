Project Bard-acious: 1.0

Medium: Coding - Unity, Dimensions: 600 x 1000


Alpha state of Bard-acious, coded in Unity to create a 2D side scroll dugeon crawler and platformer game.
Play as Archie the dragonborn bard who was left behind his teammates to fend for himself in the dugeon.
Dugeon crawl through while fighting and collecting items to help Archie advance through the stages to eventually
catch up with his teammates!

Game Rules

1. Use "a" and "d" keys to move

2. Spacebar for jumping

3. "e" key for attacking

4. "f" key for ending yourself

This game was my first time coding with Unity, while coding is not my forte, and my first time being the coder
while my partner was artist during this project. This project a was vauable learning experience of tapping
into Unity while before I only knew how to make a game through P5.js, since Unity allows me to see the overview of
how the game would function while compared to P5.js I would have to run the program to check if the scrpit was running properly.
Though it was a large learning curve on how C# works since I've never worked with C# and giving each
indivudal object their own script so every item functions in the right place.
