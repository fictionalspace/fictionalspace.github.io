Foraging Woods

Medium: Digital Art - Procreate, Photoshop, Code - Javascript and P5.JS, Dimensions: 600 x 1000S

Foraging Woods is a short web browser game created with JavaScript and P5.JS where you play as
Mabel to forage items to advance through the next level as you follow her little tale on her forest adventure.

Game Rules

1. Use the mouse to control Mabel

2. Collect items that pop up with Mabel

3. When cutscene dialogue pop up, just click to advance

4. Remember to have fun

If the demo is bit glitchy, just start moving the mouse crazy which should help in collecting the item and progressing.

Concept

I wanted to create my first playable game with the simple mechanic of collecting
while conveying a story to go along with the game. So a forest and foraging theme
was established while researching what would be foragable while keeping up with the teheme
Backgrounds were created in Photoshop while the character sprites and items were drawn using Procreate.
Coding was not my forte, so it was learning curve, but as long as I pace myself to make sure the game
was finished that it can be played then it was a good learning experience.

From Beta to Final Start Screen

Initially the concept was just a simple collecting game in the forest where you drag your mouse to the berries to collect. The simple thumbnail of the basket of berries and the
abstract bush and berry background with the font appearing like a forest was meant to convet that. Created through Photoshop while also experimenting with the brushes, since by the time
I created the final thumbnail for the demo, I have familiar myself with Photoshop and had a more concrete idea for what aesthetics and gameplay elements I wanted to convey.
